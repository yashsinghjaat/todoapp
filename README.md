# To-Do List App

A simple, modern to-do list web application with CRUD functionality that stores tasks in the browser's local storage.

## Features

- **Create**: Add new tasks
- **Read**: View all tasks
- **Update**: Edit existing tasks (double-click or use Edit button)
- **Delete**: Remove tasks
- **Mark Complete**: Check off completed tasks
- **Persistent Storage**: Tasks are saved in localStorage and persist between sessions

## How to Use

1. Open `index.html` in your web browser
2. Enter a task in the input field and click "Add Task" or press Enter
3. Double-click on a task text to edit it inline
4. Use the Edit button to start editing
5. Check the checkbox to mark tasks as completed
6. Click Delete to remove a task

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Browser Support

Works in all modern browsers that support localStorage.
